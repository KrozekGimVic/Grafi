Instalation and running tests

Run ``` git clone git@g.nosp@m.ithu.nosp@m.b.com:KrozekGimVic/Grafi.git ``` Go to root folder of the repository and do the usual compile & run cycle: ``` mkdir -p build cd build cmake .. make ../bin/tests ``` If all tests appear green, you are good to go.
